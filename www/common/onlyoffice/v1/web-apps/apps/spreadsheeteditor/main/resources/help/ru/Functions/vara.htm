Функция VARA (ДИСПА)

Функция VARA - это одна из статистических функций. Анализирует диапазон данных и возвращает дисперсию по выборке.

Синтаксис функции VARA:

VARA(argument-list)

где argument-list (список значений) - это диапазон значений, введенных вручную или находящихся в ячейках, на которые даются ссылки.

Примечание: текстовые значения и значения FALSE (ЛОЖНО) принимаются равными 0, значения TRUE (ИСТИННО) принимаются равными 1, пустые ячейки игнорируются.

Чтобы применить функцию VARA,

  1. выделите ячейку, в которой требуется отобразить результат,
  2. щелкните по значку Вставить функцию , расположенному на верхней панели инструментов,
    или щелкните правой кнопкой мыши по выделенной ячейке и выберите в меню команду Вставить функцию,
    или щелкните по значку перед строкой формул,
  3. выберите из списка группу функций Статистические,
  4. щелкните по функции VARA,
  5. введите требуемые аргументы, разделяя их запятыми, или выделите мышью диапазон ячеек,
  6. нажмите клавишу Enter.

Результат будет отображен в выбранной ячейке.
