Функция AVEDEV (СРОТКЛ)

Функция AVEDEV - это одна из статистических функций. Она используется для анализа диапазона данных и возвращает среднее абсолютных значений отклонений чисел от их среднего значения.

Синтаксис функции AVEDEV:

AVEDEV(argument-list)

где argument-list (список аргументов) - до 30 числовых значений, введенных вручную или находящихся в ячейках, на которые даются ссылки.

Чтобы применить функцию AVEDEV,

  1. выделите ячейку, в которой требуется отобразить результат,
  2. щелкните по значку Вставить функцию , расположенному на верхней панели инструментов,
    или щелкните правой кнопкой мыши по выделенной ячейке и выберите в меню команду Вставить функцию,
    или щелкните по значку перед строкой формул,
  3. выберите из списка группу функций Статистические,
  4. щелкните по функции AVEDEV,
  5. введите требуемые аргументы, разделяя их запятыми, или выделите мышью диапазон ячеек,
  6. нажмите клавишу Enter.

Результат будет отображен в выбранной ячейке.
