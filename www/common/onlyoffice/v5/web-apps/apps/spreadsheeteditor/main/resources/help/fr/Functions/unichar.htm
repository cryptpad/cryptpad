Fonction UNICAR

La fonction UNICAR est l'une des fonctions de texte et de données. Est utilisé pour renvoyer le caractère Unicode référencé par la valeur numérique donnée.

La syntaxe de la fonction UNICAR est :

UNICAR(number)

où number est le nombre Unicode qui représente le caractère. Les données peuvent être saisies à la main ou incluses dans la cellule à laquelle il est fait référence.

Pour appliquer la fonction UNICAR,

  1. sélectionnez la cellule où vous souhaitez afficher le résultat,
  2. cliquez sur l'icône Insérer une fonction située sur la barre d'outils supérieure,
    ou cliquez avec le bouton droit sur la cellule sélectionnée et sélectionnez l'option Insérer une fonction depuis le menu,
    ou cliquez sur l'icône située sur la barre de formule,
  3. sélectionnez le groupe de fonctions Texte et données depuis la liste,
  4. cliquez sur la fonction UNICAR,
  5. saisissez l'argument requis,
  6. appuyez sur la touche Entrée.

Le résultat sera affiché dans la cellule sélectionnée.
