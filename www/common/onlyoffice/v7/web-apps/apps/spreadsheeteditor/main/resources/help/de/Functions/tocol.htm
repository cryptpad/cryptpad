ZUSPALTE-Funktion

Die ZUSPALTE-Funktion ist eine der Suchen- und Bezügefunktionen. Sie wird verwendet, um das Array als eine Zeile zurückzugeben.

Bitte beachten Sie, dass dies eine Matrixformel ist. Um mehr zu erfahren, lesen Sie bitte den Artikel Matrixformeln einfügen.

Die Formelsyntax der Funktion ZUSPALTE ist:

ZUSPALTE (array, [ignore], [scan_by_column])

dabei gilt

array ist das Array oder die Referenz, die als Spalte zurückgegeben werden soll.

ignore wird verwendet, um festzulegen, ob bestimmte Arten von Werten ignoriert werden sollen. Standardmäßig werden keine Werte ignoriert. Die folgenden Werte werden verwendet: 0, um alle Werte beizubehalten (Standard); 1, um Leerzeichen zu ignorieren; 2, um Fehler zu ignorieren; 3, um Leerzeichen und Fehler zu ignorieren.

scan_by_column wird verwendet, um das Array nach Spalte zu scannen. Standardmäßig wird das Array-Scannen zeilenweise durchgeführt. Das Scannen bestimmt, ob die Werte in Zeilen oder Spalten angeordnet sind.

Anwendung der Funktion ZUSPALTE:

  1. Wählen Sie die Zelle aus, in der Sie das Ergebnis anzeigen möchten,
  2. Klicken Sie auf das Symbol Funktion
    in der oberen Symbolleiste
    oder klicken Sie mit der rechten Maustaste in eine ausgewählte Zelle und wählen Sie die Option Funktion einfügen aus dem Menü
    oder klicken Sie auf das Symbol
    in der Bearbeitungsleiste.
  3. Wählen Sie aus der Liste die Funktionsgruppe Suchen und Bezüge.
  4. Klicken Sie auf die Funktion ZUSPALTE.
  5. Geben Sie die erforderlichen Argumente ein, indem Sie sie durch Kommas trennen.
  6. Drücken Sie die Eingabe-Taste

Das Ergebnis wird in der ausgewählten Zelle angezeigt.
