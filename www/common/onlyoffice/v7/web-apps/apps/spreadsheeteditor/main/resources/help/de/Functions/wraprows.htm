ZEILENUMBRUCH-Funktion

Die ZEILENUMBRUCH-Funktion ist eine der Suchen- und Bezügefunktionen. Sie wird verwendet, um einen Zeilen- oder Spaltenvektor nach einer bestimmten Anzahl von Werten umzubrechen.

Bitte beachten Sie, dass dies eine Matrixformel ist. Um mehr zu erfahren, lesen Sie bitte den Artikel Matrixformeln einfügen.

Die Formelsyntax der Funktion ZEILENUMBRUCH ist:

ZEILENUMBRUCH(vector, wrap_count, [pad_with])

dabei gilt

vector wird verwendet, um den Vektor oder die Referenz festzulegen, die umbrochen werden müssen.

wrap_count wird verwendet, um die maximale Anzahl von Werten für jede Zeile festzulegen.

pad_with wird verwendet, um den Wert einzustellen, mit dem das Ergebnis aufgefüllt werden soll. Standardmäßig ist #N/A.

Anwendung der Funktion ZEILENUMBRUCH:

  1. Wählen Sie die Zelle aus, in der Sie das Ergebnis anzeigen möchten,
  2. Klicken Sie auf das Symbol Funktion
    in der oberen Symbolleiste
    oder klicken Sie mit der rechten Maustaste in eine ausgewählte Zelle und wählen Sie die Option Funktion einfügen aus dem Menü
    oder klicken Sie auf das Symbol
    in der Bearbeitungsleiste.
  3. Wählen Sie aus der Liste die Funktionsgruppe Suchen und Bezüge.
  4. Klicken Sie auf die Funktion ZEILENUMBRUCH.
  5. Geben Sie die erforderlichen Argumente ein, indem Sie sie durch Kommas trennen.
  6. Drücken Sie die Eingabe-Taste.

Das Ergebnis wird in der ausgewählten Zelle angezeigt.
