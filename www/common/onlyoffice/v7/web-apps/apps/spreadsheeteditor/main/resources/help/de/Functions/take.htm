ÜBERNEHMEN-Funktion

Die ÜBERNEHMEN-Funktion ist eine der Suchen- und Bezügefunktionen. Sie wird verwendet, um Zeilen oder Spalten vom Anfang oder Ende des Arrays zurückzugeben.

Bitte beachten Sie, dass dies eine Matrixformel ist. Um mehr zu erfahren, lesen Sie bitte den Artikel Matrixformeln einfügen.

Die Formelsyntax der Funktion ÜBERNEHMEN ist:

ÜBERNEHMEN(array, rows, [columns])

dabei gilt

array wird verwendet, um das Array festzulegen, aus dem Zeilen oder Spalten entnommen werden sollen.

rows wird verwendet, um die Anzahl der aufzunehmenden Zeilen festzulegen. Ein negativer Wert nimmt vom Ende des Arrays.

columns wird verwendet, um die Anzahl der zu übernehmenden Spalten festzulegen. Ein negativer Wert nimmt vom Ende des Arrays.

Anwendung der Funktion ÜBERNEHMEN:

  1. Wählen Sie die Zelle aus, in der Sie das Ergebnis anzeigen möchten.
  2. Klicken Sie auf das Symbol Funktion
    in der oberen Symbolleiste
    oder klicken Sie mit der rechten Maustaste in eine ausgewählte Zelle und wählen Sie die Option Funktion einfügen aus dem Menü
    oder klicken Sie auf das Symbol
    in der Bearbeitungsleiste.
  3. Wählen Sie aus der Liste die Funktionsgruppe Suchen und Bezüge.
  4. Klicken Sie auf die Funktion ÜBERNEHMEN.
  5. Geben Sie die erforderlichen Argumente ein, indem Sie sie durch Kommas trennen.
  6. Drücken Sie die Eingabe-Taste.

Das Ergebnis wird in der ausgewählten Zelle angezeigt.
