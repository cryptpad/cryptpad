Función JERARQUIA.EQV

La función JERARQUIA.EQV es una función estadística. Se usa para devolver el rango de un número en una lista de números. El rango de un número es su tamaño en relación con otros valores en una lista (si fuera a organizar la lista, el rango de un número sería su posición). Si más de un valor tiene el mismo rango, se devuelve el rango más alto de ese conjunto de valores.

La sintaxis de la función JERARQUIA.EQV es:

JERARQUIA.EQV(número ref[, orden])

donde

número es el valor para el que quiere encontrar el rango.

ref es el rango de celdas seleccionado que contiene el número especificado.

Orden es el valor numérico que especifica cómo ordenar el conjunto de ref. Es un argumento opcional. Si 0 se omite, la función clasifica a número como si ref fuera una lista organizada en orden descendiente. Otro valor numérico se trata como situviera el valor 1 y la función clasificanúmero como si ref fuera una lista organizada en orden ascendiente.

Para aplicar la función JERARQUIA.EQV,

  1. seleccione la celda donde usted quiere ver el resultado,
  2. pulse el icono Insertar función que se sitúa en la barra de herramientas superior,
    o haga clic derecho en la сelda seleccionada y elija la opción Insertar función en el menú,
    o pulse el icono que se sitúa en la barra de fórmulas,
  3. seleccione el grupo de funciones Estadísticas en la lista,
  4. haga clic en la función JERARQUIA.EQV,
  5. introduzca los argumentos correspondientes separados por comas,
  6. pulse el botón Enter.

El resultado se mostrará en la celda elegida.
