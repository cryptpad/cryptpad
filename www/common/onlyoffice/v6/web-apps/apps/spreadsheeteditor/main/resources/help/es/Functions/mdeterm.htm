Función MDETERM

La función MDETERM es una función matemática y trigonométrica. Se usa para devolver el determinante de una matriz de un conjunto.

La sintaxis de la función MDETERM es:

MDETERM(conjunto)

donde conjunto es un conjunto de números.

Nota: Si cualquier celda del conjunto esta vacía o contiene valores no numéricos, la función devolverá el error #N/A.
Si el número de filas en el conjunto no es igual al número de columnas, la función devolverá el error #VALUE!.

Para aplicar la función MDETERM,

  1. seleccione la celda donde usted quiere ver el resultado,
  2. pulse el icono Insertar función que se sitúa en la barra de herramientas superior,
    o haga clic derecho en la сelda seleccionada y elija la opción Insertar función en el menú,
    o pulse el icono que se sitúa en la barra de fórmulas,
  3. seleccione el grupo de función Matemáticas y trigonometría en la lista,
  4. haga clic en la función MDETERM,
  5. seleccione un rango de celdas con el ratón o introduzca los argumentos requeridos manualmente de tal modo: A1:B2,
  6. pulse el botón Enter.

El resultado se mostrará en la celda elegida.
