TEXTTEILEN-Funktion

Die Funktion TEXTTEILEN ist eine der Text- und Datenfunktionen. Sie wird verwendet, um Textzeichenfolgen durch Spalten- und Zeilentrennzeichen zu trennen.

Bitte beachten Sie, dass dies eine Matrixformel ist. Um mehr zu erfahren, lesen Sie bitte den Artikel Matrixformeln einfügen.

Die Formelsyntax der Funktion TEXTTEILEN ist:

TEXTTEILEN(text,col_delimiter,[row_delimiter],[ignore_empty], [match_mode], [pad_with])

dabei gilt

text wird verwendet, um den Text festzulegen, den Sie teilen möchten.

col_delimiter ist ein optionales Argument. Es wird verwendet, um den Text festzulegen, der den Punkt markiert, an dem der Text über Spalten verteilt werden soll.

row_delimiter ist ein optionales Argument. Es wird verwendet, um den Text festzulegen, der den Punkt markiert, an dem der Text über Zeilen verteilt werden soll.

ignore_empty ist ein optionales Argument. Es wird verwendet, um FALSE anzugeben, um eine leere Zelle zu erstellen, wenn zwei Trennzeichen aufeinanderfolgend sind. Der Standardwert ist TRUE, wodurch eine leere Zelle erstellt wird.

match_mode ist ein optionales Argument. Es wird verwendet, um den Text nach einem übereinstimmenden Trennzeichen zu durchsuchen. Standardmäßig wird eine Übereinstimmung zwischen Groß- und Kleinschreibung durchgeführt.

pad_with wird verwendet, um den Wert festzulegen, mit dem das Ergebnis aufgefüllt werden soll. Standardmäßig ist #N/A.

Anwendung der Funktion TEXTTEILEN:

  1. Wählen Sie die Zelle aus, in der Sie das Ergebnis anzeigen möchten,
  2. Klicken Sie auf das Symbol Funktion
    in der oberen Symbolleiste
    oder klicken Sie mit der rechten Maustaste in eine ausgewählte Zelle und wählen Sie die Option Funktion einfügen aus dem Menü
    oder klicken Sie auf das Symbol
    in der Bearbeitungsleiste.
  3. Wählen Sie aus der Liste die Funktionsgruppe Text und Daten.
  4. Klicken Sie auf die Funktion TEXTTEILEN.
  5. Geben Sie die erforderlichen Argumente ein, indem Sie sie durch Kommas trennen.
  6. Drücken Sie die Eingabe-Taste.

Das Ergebnis wird in der ausgewählten Zelle angezeigt.
