Функция XNPV (ЧИСТНЗ)

Функция XNPV - это одна из финансовых функций. Используется для вычисления чистой приведенной стоимости инвестиции исходя из указанной процентной ставки и нерегулярных выплат.

Синтаксис функции XNPV:

XNPV(rate, values, dates)

где

rate - это ставка дисконтирования инвестиции.

values - это массив, содержащий суммы дохода (положительные значения) или выплат (отрицательные значения). Хотя бы одно из значений должно быть отрицательным, а хотя бы одно - положительным.

dates - это массив, содержащий даты поступления или выплаты денежных средств.

Значения могут быть введены вручную или находиться в ячейке, на которую дается ссылка.

Чтобы применить функцию XNPV,

  1. выделите ячейку, в которой требуется отобразить результат,
  2. щелкните по значку Вставить функцию , расположенному на верхней панели инструментов,
    или щелкните правой кнопкой мыши по выделенной ячейке и выберите в меню команду Вставить функцию,
    или щелкните по значку перед строкой формул,
  3. выберите из списка группу функций Финансовые,
  4. щелкните по функции XNPV,
  5. введите требуемые аргументы, разделяя их запятыми,
  6. нажмите клавишу Enter.

Результат будет отображен в выделенной ячейке.
