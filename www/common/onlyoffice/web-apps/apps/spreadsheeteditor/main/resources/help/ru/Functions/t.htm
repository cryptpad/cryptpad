Функция T

Функция T - это одна из функций для работы с текстом и данными. Используется для проверки, является ли значение в ячейке (или используемое как аргумент) текстом или нет. Если это не текст, функция возвращает пустой результат. Если значение/аргумент является текстом, функция возвращает это же текстовое значение.

Синтаксис функции T:

T(value)

где value (значение) - это данные, введенные вручную или находящиеся в ячейке, на которую дается ссылка.

Чтобы применить функцию T,

  1. выделите ячейку, в которой требуется отобразить результат,
  2. щелкните по значку Вставить функцию , расположенному на верхней панели инструментов,
    или щелкните правой кнопкой мыши по выделенной ячейке и выберите в меню команду Вставить функцию,
    или щелкните по значку перед строкой формул,
  3. выберите из списка группу функций Текст и данные,
  4. щелкните по функции T,
  5. введите требуемый аргумент,
  6. нажмите клавишу Enter.

Результат будет отображен в выбранной ячейке.

Например:

Здесь один аргумент: value = A1, где A1 имеет значение дата и время. Следовательно, функция возвращает значение дата и время.

Если изменить данные в ячейке A1 с текстового на числовое значение, функция вернет пустой результат.
