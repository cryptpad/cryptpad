Функция EXACT (СОВПАД)

Функция EXACT - это одна из функций для работы с текстом и данными. Она используется для сравнения данных в двух ячейках. Функция возвращает значение TRUE (ИСТИНА), если данные совпадают, и FALSE (ЛОЖЬ), если нет.

Синтаксис функции EXACT:

EXACT(text1, text2)

где text1(2) (текст1(2)) - это данные, введенные вручную или находящиеся в ячейке, на которую дается ссылка.

Чтобы применить функцию EXACT:

  1. выделите ячейку, в которой требуется отобразить результат,
  2. щелкните по значку Вставить функцию , расположенному на верхней панели инструментов,
    или щелкните правой кнопкой мыши по выделенной ячейке и выберите в меню команду Вставить функцию,
    или щелкните по значку перед строкой формул,
  3. выберите из списка группу функций Текст и данные,
  4. щелкните по функции EXACT,
  5. введите требуемые аргументы, разделяя их запятыми,

    Примечание: функция EXACT учитывает регистр.

  6. нажмите клавишу Enter.

Результат будет отображен в выбранной ячейке.

Здесь два аргумента: text1 = A1; text2 = B1, где A1 имеет значение МойПароль, B1 имеет значение мойпароль. Следовательно, функция возвращает значение FALSE (ЛОЖЬ).
